# ABSTRACT

The Indian legal ecosystem faces persistent challenges including delayed case processing, fragmented evidence management, limited access to legal resources for citizens, and the absence of unified digital tools for law enforcement. According to the National Judicial Data Grid, over 4.5 crore cases remain pending across Indian courts, highlighting the urgent need for technology-driven solutions in the legal domain.

This project presents the design, development, and deployment of **AI Legal Assistant**, a comprehensive web-based platform built using Microsoft Blazor Server on the .NET 10.0 framework, integrated with Azure AI Foundry services. The platform serves four distinct user categories — Citizens, Lawyers, Police Officers, and System Administrators — each provided with role-specific dashboards and toolsets tailored to their operational requirements.

For **Citizens**, the platform offers conversational AI-powered case filing, automated First Information Report (FIR) generation with applicable Indian Penal Code (IPC) and Bharatiya Nyaya Sanhita (BNS) section detection, bank account freeze request generation, an emergency SOS system with GPS-based location sharing, a community-driven scam detection hub, and a cybercrime reporting portal. **Lawyers** gain access to a precedent search engine with landmark case references, a comprehensive Indian law database spanning the IPC, BNS, IT Act 2000, CrPC, BNSS, and the Constitution, and AI-assisted case management through natural language commands. **Police Officers** are equipped with an AI-powered investigation copilot (CaseIQ) that generates actionable suggestions, a phone intelligence dashboard aggregating data from telecom, banking, OSINT, and police databases through API integration, a Call Detail Record (CDR) analysis engine with pattern detection and location clustering, a blockchain-inspired evidence chain-of-custody system with hash-based integrity verification, a BNSS statutory deadline tracker with automated alerts, and a legal notice generation system supporting nine distinct notice types. **Administrators** oversee user verification workflows for Police and Lawyer accounts, ensuring only authenticated professionals access restricted modules.

The system implements multi-language support across 12 Indian languages with approximately 1,976 translation entries, dark and light theme modes with high-contrast accessibility options, and professional PDF document generation using the QuestPDF library. The Phone Intelligence module incorporates a service-oriented API architecture designed for integration with government data sources such as TAFCOP, CCTNS, and FIU-IND through secure inter-agency communication channels.

Experimental evaluation demonstrates the platform's capability to reduce manual FIR drafting time by an estimated 60-70%, streamline evidence tracking with tamper-proof custody chains, and provide investigators with AI-generated leads that consolidate multi-source intelligence into coherent analytical outputs. The modular service-oriented architecture comprises 30 independent services, 25 data model files, and 29 interactive page components, enabling independent scaling and future database migration.

**Keywords:** AI Legal Assistant, Blazor Server, Azure AI, Law Enforcement, FIR Generation, Evidence Chain of Custody, CDR Analysis, Phone Intelligence, BNSS Deadlines, Multi-language Legal Platform, Cybercrime Reporting, Role-Based Access Control
