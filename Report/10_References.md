# APPENDIX E: USER MANUAL

## E.1 System Requirements

| Requirement | Specification |
|---|---|
| Operating System | Windows 10/11, macOS 12+, or Linux |
| Browser | Google Chrome 100+, Microsoft Edge 100+, or Mozilla Firefox 100+ |
| Internet Connection | Required for Azure AI features; local features work offline |
| .NET Runtime | .NET 10.0 SDK (for self-hosting) |

## E.2 Installation Steps

1. Clone the repository:
   ```bash
   git clone https://github.com/<username>/AILegalAsst.git
   cd AILegalAsst
   ```

2. Restore NuGet packages:
   ```bash
   dotnet restore
   ```

3. Configure Azure AI (optional — fallback mode works without it):
   - Open the application configuration file
   - Update the `AzureAI.ConnectionString` with your Azure AI Foundry project connection string
   - Update `AzureAI.ModelName` if using a different model

4. Run the application:
   ```bash
   dotnet run
   ```

5. Open a browser and navigate to the application URL displayed in the terminal output.
